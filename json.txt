### **JSON (JavaScript Object Notation)**
**JSON** is a lightweight data format used for storing and exchanging data between systems, especially in web applications. It is easy to read, write, and parse by both humans and machines.

---

### **üîπ JSON Syntax**
JSON consists of **key-value pairs**, similar to JavaScript objects, but with some restrictions:
- Keys and strings **must** be enclosed in **double quotes (`""`)**.
- Data types include **strings, numbers, booleans, arrays, objects, and `null`**.

```json
{
  "name": "Alice",
  "age": 25,
  "isStudent": false,
  "hobbies": ["reading", "gaming", "traveling"],
  "address": {
    "city": "New York",
    "country": "USA"
  }
}
```

---

### **üîπ JSON vs JavaScript Object**
| Feature         | JSON                              | JavaScript Object                 |
|---------------|--------------------------------|--------------------------------|
| **Key Quotes** | **Required (`"key"`)**         | Optional (`key` or `"key"`)     |
| **Comments**   | ‚ùå Not Allowed                  | ‚úÖ Allowed (`//` or `/* */`)     |
| **Functions**  | ‚ùå Not Supported                | ‚úÖ Supported                    |

#### ‚úÖ **Example**
‚úÖ **Valid JSON**:
```json
{ "name": "Alice" }
```
‚ùå **Invalid JSON** (missing quotes on key):
```json
{ name: "Alice" }
```

---

### **üîπ JSON Data Types**
| Type       | Example |
|------------|----------------------------------|
| **String**  | `"name": "Alice"`               |
| **Number**  | `"age": 25`                     |
| **Boolean** | `"isStudent": false`            |
| **Array**   | `"hobbies": ["reading", "gaming"]` |
| **Object**  | `"address": { "city": "NY" }`   |
| **Null**    | `"nickname": null`              |

---

### **üîπ How to Work with JSON in JavaScript?**

#### **1Ô∏è‚É£ Convert JSON String to JavaScript Object**
Use `JSON.parse()` to convert a JSON string into an object:
```javascript
const jsonString = '{"name": "Alice", "age": 25}';
const obj = JSON.parse(jsonString);
console.log(obj.name); // Output: Alice
```

#### **2Ô∏è‚É£ Convert JavaScript Object to JSON String**
Use `JSON.stringify()` to convert an object into a JSON string:
```javascript
const person = { name: "Bob", age: 30 };
const jsonString = JSON.stringify(person);
console.log(jsonString); // Output: '{"name":"Bob","age":30}'
```

### **üîπ Where is JSON Used?**
‚úÖ **APIs & Web Services** ‚Üí REST APIs return data in JSON format  
‚úÖ **Configuration Files** ‚Üí `.json` files for settings (e.g., `package.json`)  
‚úÖ **LocalStorage / SessionStorage** ‚Üí Store data in the browser  
‚úÖ **Databases** ‚Üí NoSQL databases like MongoDB store data as JSON  

---

### **üöÄ Summary**
- JSON is a **text-based format** for data exchange.
- Uses **key-value pairs** and supports **strings, numbers, booleans, arrays, objects, and `null`**.
- Can be **parsed (`JSON.parse()`)** and **stringified (`JSON.stringify()`)** in JavaScript.
- Commonly used in **APIs, configurations, and databases**.

### **ðŸ”¹ How to Create a JavaScript Promise?**  
A **Promise** in JavaScript is used to handle **asynchronous operations** like API calls, file reading, or database queries.  

A Promise has three states:  
1. **Pending** â†’ Initial state  
2. **Resolved (Fulfilled)** â†’ Operation successful  
3. **Rejected** â†’ Operation failed  

---

### **ðŸ”¹ Creating a Promise**
A Promise is created using the `new Promise()` constructor.  
It takes a function with **two parameters: `resolve` and `reject`**.

```js
const myPromise = new Promise((resolve, reject) => {
    let success = true;  // Change to false to test rejection

    setTimeout(() => {
        if (success) {
            resolve("âœ… Operation Successful!");
        } else {
            reject("âŒ Operation Failed!");
        }
    }, 2000); // Simulates a delay of 2 seconds
});
```
---

### **ðŸ”¹ Handling the Promise**
You can handle a Promise using `.then()`, `.catch()`, and `.finally()`.

```js
myPromise
    .then((result) => {
        console.log(result);  // If resolved
    })
    .catch((error) => {
        console.log(error);  // If rejected
    })
    .finally(() => {
        console.log("ðŸ”„ Promise completed!");
    });
```
---

### **ðŸ”¹ Creating a Function That Returns a Promise**
```js
function fetchData() {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            let data = { user: "John Doe", age: 25 };

            if (data) {
                resolve(data);
            } else {
                reject("âŒ No Data Found!");
            }
        }, 1500);
    });
}

fetchData()
    .then((data) => console.log("User Data:", data))
    .catch((error) => console.log(error));
```
---

### **ðŸ”¹ Using `async/await` with Promises**
Instead of `.then()` and `.catch()`, you can use `async/await`.

```js
async function getUser() {
    try {
        let user = await fetchData();
        console.log("User:", user);
    } catch (error) {
        console.log(error);
    }
}

getUser();
```
âœ… **More readable and recommended for modern JavaScript!**

---

### **ðŸ”¹ Summary**
- **A Promise** represents a future value (success or failure).  
- **`resolve(value)`** â†’ Success, returns a value.  
- **`reject(error)`** â†’ Failure, returns an error.  
- **`.then()`** â†’ Runs when resolved.  
- **`.catch()`** â†’ Runs when rejected.  
- **`.finally()`** â†’ Runs after `.then()` or `.catch()`.  
- **Use `async/await`** to handle Promises in a clean way.
